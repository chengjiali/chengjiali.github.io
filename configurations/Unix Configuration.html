Compilation and Installation

Python2升级到Python3无法导入包：

Magic Number Error详见 https://stackoverflow.com/questions/514371/whats-the-bad-magic-number-error

Conda安装R

conda create -n r -c r r-essentials
conda install -c intel tcl=8.6.4

源码安装软件：

  1. wget获取源码
  2. 解压 tar -zxvf mysql-5.5.28.tar.gz
  3. Configure and Make

编译安装tcl/tk

http://blog.csdn.net/cryhelyxx/article/details/22514871
–with-tcl= –prefix=
https://docs.python.org/3/library/tkinter.html

Network

Change MAC Address

sudo ifconfig en0 MAC ADDR

SSH配置：

  1. 在服务器上安装server
  2. 在client上生成秘钥
  3. 将client秘钥存储在服务器文件内
  4. 重启服务器

如果无法以root用户登录:

  1. 更改root用户密码 sudo passwd root

Curl for HTTP2 support:

需要重新编译curl
需要nghttp2和任意一种SSL支持
https://curl.haxx.se/docs/http2.html
./configure –prefix=/Users/CJL-RMBP/curl-http2-support/ –with-nghttp2=/usr/local/Cellar/nghttp2/1.30.0 –with-ssl=/usr/local/Cellar/openssl/1.0.2j
